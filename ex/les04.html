Поиск и замена значений в данных

Два способа замены:

  1. Макрос - при помощи кода VBA
  2. PowerQuery - мощный инструмент для загрузки и предварительной обработки данных

Замена при помощи макроса

  1. Включим запись макроса
  2. Выполним поиск и замену обычным образом.
    1. Например, заменим "улица" на "ул." при помощи поиска и замены
  3. Остановим запись макроса, и найдем ёго код, автоматически сгенерированный Excel, в редакторе VBA
  4. Используйте этот код в качестве образца. Добавьте в него необходимые изменения.
    1. Например, установите диапазон для поиска и замены
    2. Добавьте значения для поиска и замены
  5. Добавьте кнопку на лист (вкладка Разработчик > Вставка) и привяжите к ней макрос для удобного запуска.

Работа с данными листа - чтение

Разные способы получить данные из листа:

  1. Свойство Selection - текущие выделенные ячейки
  2. Свойство Range - поиск диапазона по адресу (номера строк и столбцов)
  3. Именованный диапазон

Итерация по значениям: при помощи цикла For Each

Для отладки кода используйте следующие методы:

  • пошаговое выполнение: меню Debug, команды Step Into, Step Over, Step Out
  • точки останова - Breakpoint
  • окно Locals для просмотра значений переменных
  • окно Immediate для выполнения команд, вычисления значений и отладочных сообщений

Работа с данными листа - запись

  1. Находим диапазон, в который хотим записать данные, при помощи Range, именованного диапазона, и т.п.
  2. Адресация отдельных ячеек в диапазоне при помощи свойства Cells
  3. Смещение ячейки при помощи метода Offset
    1. Сместиться в конец области данных можно при помощи метода End, например: Range("A1").End(xlDown).Offset(1, 0)
  4. Запись значения через свойство ячейки Value

Исследовать доступные объекты VBA, их свойства и методы можно при помощи Object Browser.

Функции VBA для работы со строками

Следующие функции могут пригодиться при работе со строками. См. также список функций на сайте Microsoft.

  1. InStr:

    • Используется для поиска позиции подстроки в строке. Если подстрока найдена, возвращает ее начальную позицию, иначе – 0.
    • Пример: InStr(address, "д.") – находит, на какой позиции в address находится "д.".

  2. Left:

    • Извлекает указанное количество символов с левого края строки.
    • Пример: Left(address, housePos - 1) – берет часть строки address от начала до housePos - 1, исключая символ "д.".

  3. Mid:

    • Извлекает подстроку, начиная с заданной позиции и длиной в указанное количество символов.
    • Пример: Mid(address, housePos + 3, apartmentPos - housePos - 3) – берет подстроку, начиная с housePos + 3, пропуская "д. ", и заканчивая перед apartmentPos.

  4. Right:

    • Возвращает последние N символов строки.
    • Пример: Right(address, Len(address) - apartmentPos - 3) – извлекает часть строки address, начиная с позиции apartmentPos + 3 и до конца.

  5. Trim:

    • Удаляет пробелы в начале и конце строки.
    • Пример: Trim(street) – убирает лишние пробелы в street.

  6. Len:

    • Возвращает количество символов в строке, включая пробелы.
    • Пример: Len(address) – дает длину строки address.

  7. Replace:

    • Позволяет заменить одну подстроку на другую в заданной строке.
    • Пример: Replace(address, " ", " ") – заменяет двойные пробелы на одиночные, убирая лишние пробелы.
