Работа с данными в Excel и VBA

Техники для таблиц Excel

  1. Импорт данных с веб-страниц:

    • Импорт данных с веб-страницы через вкладку "Данные" > "Из Интернета".

    • Редактор Power Query позволяет загружать, очищать и трансформировать данные перед их импортом в Excel, облегчая работу с наборами данных.

    • Данные можно обновлять после загрузки.

  2. Форматирование таблиц:

    • Применение стилей к таблицам - не только визуальное оформление, но создание объекта таблицы.
    • Автоматическая ширина столбца при двойном щелчке на границе между названиями.

  3. Горячие клавиши:

    • Ctrl + Space — выделение столбца.
    • Shift + Arrow Keys — для выбора нескольких столбцов (влево-вправо).
    • Ctrl + A — для выделения текущего блока, всей таблицы или данных (нажимать последовательно).
      • стрелки с нажатым Ctrl - прыжок в конец блока
      • с нажатым Shift - выделение ячеек
    • Ctrl + C и Ctrl + V — для копирования и вставки.

Техники VBA

  1. Запись макросов для автоматизации действий.

    • Для запуска записи макроса используйте "Разработчик" > "Запись макроса", либо Alt + T + M + R.
    • Просмотр и редактирование VBA-кода.
    • Чтобы увидеть список макросов и запустить их, используйте "Разработчик" > "Макросы" или нажмите Alt + F8.

  2. Доступ к редактору VBA:

    • В редактор кода можно перейти через редактирование макроса либо сочетания клавиш Alt + F11.
    • В окне проекта (Project Explorer) выберите необходимый лист, модуль или форму.
    • Добавление нового модуля через "Вставка" > "Модуль".
    • Код в редакторе можно запустить с помощью F5.
