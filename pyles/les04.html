Разбор ДЗ и пройденного материала

  1. Приоритет операций и скобки:

    • Операции в Python выполняются в определенном порядке. Например, умножение и деление выполняются перед сложением и вычитанием.
    • Скобки используются для изменения порядка выполнения операций.
    • При вызове функции сначала выполняются операции внутри скобок (аргументы), а затем результаты этих операций используются

    Пример:

    print(type(int('1')))  # сначала вызовется функция int с аргументом '1',
                       # затем type с аргументом 1,
                       # затем результат будет выведен на экран функцией print

  2. Переменное количество аргументов функции:

    • Функции могут иметь фиксированное либо переменное число аргументов
    • Например, функция input() всегда имеет один аргумент, а функция print() может вызываться с любым количеством аргументов

    Пример:

    # Функция с фиксированным количеством аргументов
x = input('Введите значение x: ')

# Функция с переменным количеством аргументов
print()
print(x)
print(1, 2, 3, 4)

  3. Использование переменных для часто повторяющихся выражений:

    Переменные помогают избежать дублирования кода и делают его более читаемым.

    Пример:

    base_url = "https://example.com"
url1 = base_url + "/page1"
url2 = base_url + "/page2"
print(url1)
print(url2)

  4. Тип float:

    Тип float используется для представления чисел с плавающей запятой (десятичных).

    Пример:

    pi = 3.14159
radius = 5.0
area = pi * radius ** 2
print(type(area)) # Вывод: <class 'float'>

Типы коллекций в Python

  1. Кортежи:

    Кортежи — это неизменяемые последовательности элементов.

    Пример:

    my_tuple = (1, 2, 3, 4)
print(type(my_tuple)) # Вывод: <class 'tuple'>

  2. Списки:

    Списки — это изменяемые последовательности элементов. (На следующем уроке изучим как добавлять и удалять элементы из списка.)

    Пример:

    my_list = [1, 2, 3, 4]
print(type(my_list)) # Вывод: <class 'list'>

  3. Итерация — перебор элементов коллекции:

    Цикл for:

    Используется для перебора элементов коллекции.

    Пример:

    # Цикл for для перебора элементов списка
for item in my_list:
   print(item)

    Отступы важны в Python для определения блоков кода.
