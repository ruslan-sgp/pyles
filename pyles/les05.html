Пример решения:

# Список стран
countries = [
    "Japan",
    "Brazil",
    "Russia",
    "Canada",
    "Germany",
    "Australia",
    "India",
    "United States",
    "France"
]

# Вывод пронумерованного списка
line_number = 1
for country in countries:
    print(f"{line_number}. {country}")
    line_number = line_number + 1

Обратите внимание:

  • в этом примере мы используем переменную line_number для вывода номера строки

  • в каждой итерации цикла мы увеличиваем значение line_number на 1

  • увеличение значения line_number на 1 происходит после вывода страны, чтобы номер строки соответствовал выводу

  • выражение, присваемое переменной line_number, содержит эту переменную, что позволяет ей увеличиваться на 1 в каждой итерации цикла

  • используется f-строка (интерполяция строк) для вывода номера и названия страны: f"{index}. {country}"

  • можно также использовать конкатенацию строк, результат будет тот же:

    # А) склеить строку и передать единым аргументом
print(str(line_number) + ". " + country)

# Б) передать два аргумента
print(str(line_number) + ".", country)

  • в этом случае мы используем str(line_number) для преобразования line_number в строку, чтобы добавить к номеру точку

  • имена переменных могут содержать цифры и знак подчеркивания, но не должны начинаться с цифры:

    # правильно

a1 = 10
a2 = 20

b_1 = 5
line_10 = 'hello'

# неправильно

1a = 10 # ошибка! имена переменных не могут начинаться с цифры

Что нового мы узнали

Коллекции - списки и кортежи

  1. Доступ к элементам по индексу

    • Как списки, так и кортежи позволяют доступ к элементам по номеру, который называется индексом.
    • Пример: 
      my_list = [1, 2, 3]
my_tuple = (10, 20, 30)
print(my_list[0])  # Вывод: 1
print(my_tuple[1])  # Вывод: 20

  2. Индексация начинается с 0

    • В Python индексация начинается с 0, что означает, что первый элемент имеет индекс 0.

    Пример:

    print(my_list[0])  # Первый элемент списка
print(my_tuple[0])  # Первый элемент кортежа

  3. Изменение элемента по индексу

    • Списки являются изменяемыми (mutable), и вы можете изменять элементы по индексу.
    • Кортежи являются неизменяемыми (immutable), и попытка изменить элемент приведет к ошибке.

    Пример:

    my_list[0] = 10  # Изменение первого элемента списка
# my_tuple[0] = 10  # Это вызовет ошибку TypeError

  4. Метод append

    • Списки имеют метод append(), который позволяет добавлять элементы в конец списка.
    • Кортежи не имеют метода append(), и попытка его использовать приведет к ошибке.
    • Пример: 
      my_list.append(5)  # Добавление элемента в список
# my_tuple.append(5)  # Это вызовет ошибку AttributeError
    • Можно создать пустой список, затем добавить в него элементы: 
      my_list = []
my_list.append(1)
my_list.append(2)
my_list.append(3)
print(my_list)  # Вывод: [1, 2, 3]

  5. Конкатенация

    • Оба типа коллекций поддерживают конкатенацию с помощью оператора +. Пример: 
      list1 = [1, 2]
list2 = [3, 4]
combined_list = list1 + list2  # [1, 2, 3, 4]

tuple1 = (1, 2)
tuple2 = (3, 4)
combined_tuple = tuple1 + tuple2  # (1, 2, 3, 4)

  6. Ошибка TypeError при попытке изменить элемент кортежа

    • Если вы попытаетесь изменить элемент кортежа, то получите ошибку TypeError.

    Пример:

    # my_tuple[0] = 10  # Это вызовет TypeError: 'tuple' object does not support item assignment

  7. IndexError

    • Если вы попытаетесь получить доступ к элементу по индексу, который выходит за пределы длины коллекции, вы получите ошибку IndexError.

    Пример:

    print(my_list[10])  # Это вызовет IndexError: list index out of range
print(my_tuple[10])  # Это также вызовет IndexError: tuple index out of range

  8. AttributeError при попытке добавить элемент в кортеж

    • Если вы попытаетесь использовать метод append() на кортеже, вы получите ошибку AttributeError.

    Пример:

    my_tuple.append(5)  # Это вызовет AttributeError: 'tuple' object has no attribute 'append'

  9. Тип элементов коллекции

    • элементами коллекции (кортежа или списка) могут быть не только числа, но и другие данные:
    # Список с элементами разных типов: int, str, float
my_list = [1, "hello", 3.14]
print(my_list) # Вывод: [1, 'hello', 3.14]

# Кортежи с элементами разных типов: имя и возраст студента
student_1 = ("John", 14)
student_2 = ("Alice", 15)
student_3 = ("Bob", 16)
print(student_1) # Вывод: ('John', 14)

# при создании списка можно указать переменные в качестве элементов
# при этом будут скопированы значения этих переменных
a = 5
b = 10
c = 20

my_list = [a, b, c] # содержание списка: [5, 10, 20]
print(my_list) # Вывод: [5, 10, 20]

a = 10 # изменилось значение переменной, но список остался прежним
print(my_list) # Вывод: [5, 10, 20]

# создадим список с данными студентов
students = [student_1, student_2, student_3]
for student in students:
    print(student)
# Вывод:
# ('John', 14)
# ('Alice', 15)
# ('Bob', 16)

  10. Строки как коллекции

    • Строки в Python также являются коллекциями, элементами которых являются отдельные символы строки.
    • Строки неизменяемы - вы можете получить доступ к символам строки по индексу, но не можете изменять их.

    Пример:

    my_string = "Hello"
print(my_string[0])  # Вывод: H
print(my_string[1])  # Вывод: e
# my_string[0] = 'h'  # Это вызовет TypeError

Задание 1
  • нам нужно вывести элементы коллекции по отдельности, значит будем использовать цикл for
  • для начала сделаем итерацию по списку имён. Запустите следующий код и проверьте результат:
names = ('John', 'Alice', 'Bob')
ages = (14, 15, 16)

for name in names:
  print(name)
  • это выводит только список имён, теперь к каждому имени нужно добавить возраст
    • то есть, к первому имени из списка names добавить первое значение из списка ages; ко второму значению из names добавить второе значение из ages и так далее
  • номера элементов в коллекциях начинаются с 0 и увеличиваются на 1 при переходе к следующему элементу
  • создадим переменную-счётчик n с номером текущего элемента с начальным значением 0, и будем её увеличивать на 1 в каждой итерации цикла
    • так же мы делали в задании со списком стран
  • если напечатать значение n в каждой итерации, то получим индекс элемента в первой колонке:
names = ('John', 'Alice', 'Bob')
ages = (14, 15, 16)

n = 0
for name in names:
  print(n, name)
  n = n + 1

Запустите код и проверьте результат.

  • теперь используем переменную n в качестве индекса для получения элемента из ages
Решение 
names = ('John', 'Alice', 'Bob')
ages = (14, 15, 16)

n = 0
for name in names:
  print(name, ages[n])
  n = n + 1
