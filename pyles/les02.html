Урок 2

Что мы узнали

  1. Ввод данных с клавиатуры

    • Использование функции input() для получения данных от пользователя.
    • Ввод текстовых данных и чисел с сохранением в переменную.

  2. Сообщения Python об ошибках

    • Синтаксическая ошибка — SyntaxError: возникает при нарушении правил написания кода (например, пропущенная закрывающая скобка и т.п.).

  3. Конкатенация строк

    • Объединение строк при помощи оператора +.
    • Пример вывода сообщения с "склеиванием" текста и переменных.

  4. f-строки (интерполяция строк)

    • Форматирование строк с подстановкой значений переменных.
    • Упрощение вывода сложных строк.

  5. Типы данных переменных

    • Знакомство с основными типами: str (строки) и int (целые числа).
    • Определение типов переменных с помощью функции type().

  6. Преобразование типов

    • Применение функции int() для преобразования строки в число.
    • Использование функции str() для обратного преобразования - из числа в строку.
    • Ошибка типа — TypeError: возникает, когда операция или функция применяется к объекту неподходящего типа (например, сложение строки и числа).

  7. Тройные кавычки

    • Использование тройных кавычек для написания многострочного текста.
    • Пример многострочного вывода.

Пример кода:

# ========================================
#  Commentaries and input
# ========================================


# greet the user
name = input('Hello! What is your name? ')
print('Hello, ' + name + '!')

# get number of apples
a = input('How many apples do you have? ')

# get number of bananas
b = input('How many bananas do you have? ')


# ========================================
# Formatting strings
# ========================================


# show the total number of fruits
# use string concatenation
print('You have ' + a + ' apples and ' + b + ' bananas.')

# use string interpolation
print(f'In total you have {a} apples and {b} bananas.')


# ========================================
# Type conversion
# ========================================


# adds two strings together, not two numbers
total = a + b
print(f'In total you have {total} fruits (WRONG!).')

# print type information
print(f'Type of a is {type(a)}, type of b is {type(b)}')

# convert types
numApples = int(a)
numBananas = int(b)

# this will not work, because a number is added to a string
# TypeError will be thrown
# print('You have ' + numApples + ' apples') # WRONG!

print(f"""
Type of numApples is {type(numApples)}
Type of numBananas is {type(numBananas)}
""")

# it will do correct addition now
print(f'In total you have {numApples + numBananas} fruits.')
