Эксперименты с модулем Turtle в Python

Давайте познакомимся с модулем turtle в Python. Мы научимся создавать рисунки и экспериментировать с кодом.

1. Введение в Turtle

Что такое turtle?

  • turtle — это встроенный в Python модуль, который позволяет рисовать графику с помощью "черепахи", которая перемещается по экрану.
  • Это отличный инструмент для изучения основ программирования и создания визуальных эффектов.

Как добавить turtle в программу?

Есть два разных способа:

  1. используя функции из модуля turtle, или

  2. используя объект Turtle.

Способ 1 - более простой

Вставьте следующую строку в начало вашего кода:

from turtle import *   # импортировать все функции из модуля turtle

Затем используйте функции из модуля turtle, например:

forward(100)  # передвинуть черепаху на 100 единиц вперёд
left(90)      # повернуть черепаху влево на 90 градусов

... и так далее.

Способ 2 - с использованием объектов

Создайте объект черепахи с помощью Turtle(), затем используйте методы этого объекта для рисования:

from turtle import Turtle   # импортировать класс Turtle из модуля turtle
t = Turtle()                # создать объект черепахи
t.forward(100)              # передвинуть черепаху на 100 единиц вперёд
t.left(90)                  # повернуть черепаху влево на 90 градусов

Данный способ позволяет создавать более одного объекта черепахи и управлять ими по-отдельности.

Например, чтобы создать вторую черепаху, вы можете сделать следующее:

t2 = Turtle()        # создать вторую черепаху
t2.color("blue")     # изменить цвет пера второй черепахи на синий
t2.right(45)         # повернуть черепаху вправо на 45 градусов
t2.backward(200)     # передвинуть черепаху на 200 единиц назад

Примечание:

  1. методы - это функции, которые принадлежат объекту. В этом примере forward и left принадлежат объектам t и t2.
  2. t, t2 - это экземпляры (объекты) класса Turtle.
  3. Можно установить разные свойства (например, цвет, толщину пера, и другие) для каждой черепашки.

Базовые команды:

import turtle

# Создаем объект черепахи
t = turtle.Turtle()

# Движение черепахи
t.forward(100)  # Движение вперед на 100 единиц
t.right(90)     # Поворот направо на 90 градусов
t.forward(100)
t.left(45)      # Поворот налево на 45 градусов
t.backward(50)  # Движение назад на 50 единиц

# Завершение рисования
turtle.done()

4. Работа с цветами

  • Изменение цвета пера и заливки: 
    t.color("blue")          # Цвет пера
t.fillcolor("yellow")    # Цвет заливки
t.begin_fill()           # Начало заливки
t.circle(50)             # Рисование круга
t.end_fill()             # Конец заливки

5. Настройка черепахи

  • Изменение формы и скорости: 
    t.shape("turtle")  # Форма черепахи
t.speed(5)         # Скорость (1 = медленно, 10 = быстро)
t.pensize(3)       # Толщина пера

6. Координатная система черепахи

Координатная система черепахи основана на двумерной декартовой системе координат. Её понимание открывает возможности для точного управления положением и движением черепахи, а также помогает создавать графику и осваивать основы программирования. 🐢🎨

6.1. Начало координат

  • Центр экрана — это точка (0, 0), которая является началом координат.
  • По умолчанию черепаха появляется в центре экрана, направленная вправо (в положительном направлении оси X).

6.2. Оси координат

  • Ось X:

    • Горизонтальная ось.
    • Положительные значения — вправо от центра.
    • Отрицательные значения — влево от центра.

  • Ось Y:

    • Вертикальная ось.
    • Положительные значения — вверх от центра.
    • Отрицательные значения — вниз от центра.

6.3. Углы поворота

  • Направление черепахи:

    • По умолчанию черепаха направлена вправо (0 градусов).
    • Поворот вправо (right) увеличивает угол.
    • Поворот влево (left) уменьшает угол.

  • Градусы:

    • 0° — вправо.
    • 90° — вверх.
    • 180° — влево.
    • 270° — вниз.
    • 360° — снова вправо.

6.4. Примеры работы с координатами

  • Перемещение в точку (x, y):

    t.goto(100, 50)   # Переместить черепаху в точку (100, 50)
t.home()          # Возврат в начало (0, 0)

  • Рисование квадрата с использованием координат:

    t.goto(100, 0)
t.goto(100, 100)
t.goto(0, 100)
t.goto(0, 0)

6.5. Определение текущих координат

  • Метод t.position(): Возвращает текущие координаты черепахи.

    print(t.position())  # Вывод: (100.00, 50.00)

  • Метод t.xcor() и t.ycor(): Возвращают текущие координаты X и Y соответственно.

    print(t.xcor())  # Вывод: 100.00
print(t.ycor())  # Вывод: 50.00

6.6. Ориентация черепахи

  • Метод t.heading(): Возвращает текущий угол направления черепахи.

    print(t.heading())  # Вывод: 0.0 (направление вправо)

  • Установка направления:

    t.setheading(90)  # Повернуть черепаху вверх (90 градусов)

6.7. Границы экрана

  • Размер экрана можно настроить с помощью turtle.Screen().
  • По умолчанию экран имеет размеры 400x300 пикселей, но это можно изменить: 
    screen = turtle.Screen()
screen.setup(width=600, height=600)  # Установка размера экрана

6.8. Пример использования координат

import turtle

t = turtle.Turtle()

# Перемещение в разные точки
t.goto(100, 100)  # Вправо и вверх
t.goto(-100, 100) # Влево и вверх
t.goto(-100, -100)# Влево и вниз
t.goto(100, -100) # Вправо и вниз
t.goto(0, 0)      # Вернуться в центр

# Завершение
turtle.done()

7. Создание функций для повторного использования

        t.forward(size)

draw_star(100)

8. Эксперименты со случайностью

  • Использование модуля random: 
    import random

colors = ["red", "green", "blue", "purple", "orange"]
for _ in range(20):
    t.color(random.choice(colors))
    t.forward(random.randint(50, 150))
    t.right(random.randint(0, 360))

9. Создание интерактивных программ

  • Управление черепахой с клавиатуры: 
    def move_forward():
    t.forward(50)

def turn_left():
    t.left(90)

# Настройка клавиш
turtle.listen()
turtle.onkey(move_forward, "Up")
turtle.onkey(turn_left, "Left")

turtle.done()

10. Комбинирование концепций

  • Рисование спирали с изменяющимся размером: 
    def draw_spiral(size, angle):
    for _ in range(100):
        t.forward(size)
        t.right(angle)
        size += 2

draw_spiral(5, 91)

11. Дополнительные возможности

  • Настройка окна:

    screen = turtle.Screen()
screen.bgcolor("black")  # Цвет фона
screen.title("Эксперименты с Turtle")   # Заголовок окна

  • Сохранение рисунка:

    import turtle

t = turtle.Turtle()
t.circle(100)

# Сохранение рисунка в формате .eps
canvas = turtle.getcanvas()
canvas.postscript(file="circle.eps")

  • Очистка экрана:

    t.clear()               # стереть рисунок
t.reset()              # стереть рисунок и вернуть исходное состояние черепашки
turtle.clearscreen()    # очистить экран

  • Разное:

    penup()         # поднять перо
pendown()       # опустить перо
stamp()         # отпечатать черепашку
hideturtle()    # скрыть черепашку
done()          # ожидать перед закрытием экрана
exitonclick()   # закрыть окно при щелчке мышью
