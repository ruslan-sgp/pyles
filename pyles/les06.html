Графика Turtle в Python

Turtle — это встроенный модуль Python, который позволяет создавать графические изображения с помощью "черепашки", управляемой при помощи команд.

Подготовка к использованию графики Turtle

  1. Импорт модуля:

Чтобы использовать Turtle в своей программе, необходимо импортировать модуль turtle при помощи команды import:

import turtle
  • Понятие модуля: Модуль — это файл или библиотека кода, которая предоставляет дополнительные функции и классы. Turtle — один из стандартных модулей Python.
  • Импорт модуля: С помощью import мы загружаем модуль в нашу программу. После этого мы можем использовать функции и классы модуля в нашем коде.

  1. Создание черепашки:

    t = turtle.Turtle()
    • Здесь создается объект черепашки t, который мы используем для рисования.

  2. Объекты:

    • Объект: Черепашка t — это объект, который имеет свои свойства (например, цвет пера) и методы (например, forward).
    • Метод и Функция: Методы похожи на функции, но они связаны с объектом. Например, t.forward(100) — это метод объекта t "черепашка".
    • Конструктор: turtle.Turtle() — это специальный метод, который создает объект.

В целом, код для создания черепашки и начала работы с ней выглядит так:

# импорт модуля
import turtle
# создание черепашки
t = turtle.Turtle()
# вызов методов черепашки
t.forward(100)
t.left(90)

Рисование фигур при помощи Turtle

Движение и рисование линий:

t.forward(100)  # черепашка двигается вперед на 100 пикселей
t.left(90)      # поворот влево на 90 градусов
t.right(45)     # поворот вправо на 45 градусов

Использование циклов для повторяющихся действий

Квадрат:

for _ in range(4):
    t.forward(100)
    t.left(90)
  • Используем цикл for для повторения одинаковых действий.
  • здесь переменная цикла называется _. Данное имя показывает, что значение переменной не используется, но оно необходимо для цикла.

Многоугольник:

Общий случай для замкнутого многоугольника с n сторонами:

sides = 6  # количество сторон
angle = 360 / sides  # угол поворота
for _ in range(sides):
    t.forward(100)
    t.left(angle)

Рисование узоров и фракталов

Пример спирали:

colors = ['red', 'green', 'blue', 'orange', 'purple']
for i in range(50):
    t.pencolor(colors[i % len(colors)])
    t.forward(10 + i * 2)
    t.left(45)

Использование координатной системы

Черепашка может перемещаться в определенные координаты:

t.goto(50, 50)  # перемещается в точку (50, 50) по координатной плоскости

Система координат:

  • Центр холста — точка (0, 0).
  • X увеличивается вправо, Y — вверх.

Дополнительные функции и методы Turtle

  • Подъем и опускание пера: 
    t.penup()    # перестает рисовать
t.pendown()  # снова начинает рисовать
  • Цвет и ширина линии: 
    t.pencolor('blue')  # устанавливает цвет пера
t.width(5)          # устанавливает толщину линии
  • Помощь с методами: 
    help(t)             # общая информация о методах черепашки
help(t.forward)     # справка по методу forward

Константы и переменные

  • Константа: Значение, которое не изменяется во время выполнения программы.
  • Соглашения в Python: Константы принято писать заглавными буквами: 
    SIDE_LENGTH = 100  # константа для длины стороны
  • Использование констант в качестве параметров алгоритма: 
      SIDE_LENGTH = 100  # длина стороны многоугольника
  NUM_SIDES = 6      # количество сторон многоугольника
  ANGLE = 360 / NUM_SIDES  # угол поворота

  for _ in range(NUM_SIDES):
      t.forward(SIDE_LENGTH)
      t.left(ANGLE)

Новые функции Python

  • range: Создает последовательность чисел: 
    for i in range(5):  # числа от 0 до 4
    print(i)
  • len: Возвращает длину списка или строки: 
    colors = ['red', 'green', 'blue']
print(len(colors))  # вывод: 3
