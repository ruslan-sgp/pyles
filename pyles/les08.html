Урок 8

Продолжаем рисовать!

Упражнение 1

Рисуем звезду при помощи goto и координат:

from turtle import *

SIZE = 500
speed(8)

width(5)
color("red")

write(position())
goto(100, SIZE)
write(position())
goto(200, 0)
write(position())
goto(-100, 300)
write(position())
goto(300, 300)
write(position())
goto(0, 0)

done()

Результат:

Обсуждение:

  • Что такое координаты? Как они влияют на рисование?
  • Почему полезно знать текущую позицию?

Упражнение 2

Рисуем звезду при помощи forward и поворотов:

from turtle import *

width(5)
color("blue")

setheading(90)
right(18)

forward(400)
right(144)
forward(400)
right(144)
forward(400)
right(144)
forward(400)
right(144)
forward(400)

done()

Результат:

Обсуждение:

  • Почему угол поворота звезды равен 144 градусам?
  • Что произойдёт, если изменить угол?

Упражнение 3

Сокращаем код с помощью цикла:

from turtle import *

width(5)
color("green")

for _ in range(5):
    forward(400)
    right(144)

done()

Результат:

Теперь можно легко нарисовать даже несколько звёзд:

from turtle import *

# первая звезда
penup()
goto(-400, -500)
pendown()
color("blue")

setheading(90)
right(18)

for _ in range(5):
    forward(400)
    right(144)

# вторая звезда
color("violet")
penup()
goto(-400, 0)
pendown()

for _ in range(5):
    forward(400)
    right(144)

# третья звезда
penup()
goto(0, -200)
color("green")
pendown()
for _ in range(5):
    forward(300)
    right(144)

done()

Результат:

Упражнение 4

Создаем функцию для повторяющегося кода:

from turtle import *

width(5)

def star():
    for _ in range(5):
        forward(300)
        right(144)

# первая звезда
penup()
goto(-400, 0)
pendown()
color("OrangeRed")

star()

# вторая звезда
penup()
goto(-100, 300)
pendown()
color("SteelBlue")

star()

# третья звезда
penup()
goto(-100, -300)
pendown()
color("LimeGreen")

star()

done()

Результат:

Добавляем параметры в функцию:

from turtle import *

width(5)

def star(size, x, y, clr):
    penup()
    goto(x, y)
    pendown()
    color(clr)
    for _ in range(5):
        forward(size)
        right(144)

star(100, -400, 100, 'red')
star(200, -200, 100, 'blue')
star(300, 100, 100, 'green')

done()

Результат:

Теперь рисовать звёзды стало еще проще!

Что мы узнали

  • Как использовать координаты и функцию goto для рисования.
  • Как работают команды forward и right для управления черепашкой.
  • Почему полезно использовать циклы для повторяющихся действий.
  • Как создавать функции, чтобы упростить и организовать код.
  • Как добавлять параметры в функции для их большей гибкости.
