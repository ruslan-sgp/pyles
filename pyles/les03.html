Что мы узнали

NameError

  • Объявление переменных (define) перед использованием
    • Пример: Если вы попытаетесь использовать переменную x до её объявления, Python выдаст ошибку NameError. 
      print(x)  # NameError: name 'x' is not defined
  • Опечатка в названии переменной
    • Пример: Если вы объявили переменную count, но по ошибке написали coutn, Python также выдаст ошибку NameError. 
      count = 10
print(coutn)  # NameError: name 'coutn' is not defined
  • Python показывает строку и место в строке для каждой ошибки

    • Пример: Если ошибка возникла в строке 5, Python укажет на это место.

      Traceback (most recent call last):
File "d:\Dev\Code\ITClub\Python\PY02\les03\hw.py", line 5, in <module>
  print(x)
        ^
NameError: name 'x' is not defined

Важность значимых названий переменных

  • Делают код удобным для понимания
    • Пример: Использование переменной total_price вместо tp делает код более читаемым. 
      total_price = 100  # Хорошо
tp = 100  # Плохо
  • Использование комментариев, отступов, и т.п. также делает код понятным и приятным для чтения

Вывод и форматирование данных

  • Создание колонок при помощи пробелов
    • Пример: 
      print(f"Item    Price")
print(f"Apple    $1.00")
  • Вставка значений в строки
    • Переменные или выражения
      • Пример: 
        item = "Apple"
price = 1.00
print(f"{item}    ${price}")

Синтаксис: скобки, кавычки, запятые, и т.д.

  • Пример: круглые скобки при вызове функций
    • Пример: 
      print(1, 2, 3)  # Круглые скобки используются для вызова функции print
  • Возвращение результата функции и сохранение в переменной
    • Пример: 
      result = max(1, 2, 3)  # Результат функции max сохраняется в переменной result

Использование горячих клавиш в VS Code

  • Alt+Стрелки вверх и вниз для перемещения целых строк
    • Пример: Выделите строку и нажмите Alt+Стрелка вверх, чтобы переместить её на одну строку выше.
