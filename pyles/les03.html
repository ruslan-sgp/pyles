NameError

  • Объявление переменных (define) перед использованием

    • Пример: Если вы попытаетесь использовать переменную x до её объявления, Python выдаст ошибку NameError. 
      print(x)  # NameError: name 'x' is not defined

  • Опечатка в названии переменной

    • Пример: Если вы объявили переменную count, но по ошибке написали coutn, Python также выдаст ошибку NameError. 
      count = 10
print(coutn)  # NameError: name 'coutn' is not defined

  • Python показывает строку и место в строке для каждой ошибки

    • Пример: Если ошибка возникла в строке 5, Python укажет на это место. 
      Traceback (most recent call last):
File "d:\Dev\Code\ITClub\Python\PY02\les03\hw.py", line 5, in <module>
  print(x)
        ^
NameError: name 'x' is not defined

Важность значимых названий переменных

  • Названия переменных должны отражать их назначение. Это делает код понятным и удобным, упрощает работу

    • Пример: Использование переменной total_price вместо tp делает код более читаемым. 
      total_price = 100  # Хорошо
tp = 100  # Плохо

  • Использование комментариев, отступов, и т.п. также делает код понятным и приятным для чтения

    • Пример: сравним два примера кода, которые делают одно и то же, но оформлены по-разному:

      А) Плохо отформатированный код

      math_grade=85
science_grade=92
history_grade=78
total_grades=math_grade+science_grade+history_grade
average_grade=total_grades/3
print("Средний балл студента:",average_grade)

      Б) Хорошо отформатированный код

      # Оценки по предметам
math_grade = 85
science_grade = 92
history_grade = 78

# Вычисление общей суммы оценок
total_grades = math_grade + science_grade + history_grade

# Вычисление среднего балла
average_grade = total_grades / 3

# Вывод результата
print("Средний балл студента:", average_grade)

Синтаксис: скобки, кавычки, запятые, и т.д.

При написании кода важно соблюдать правильный синтаксис, чтобы избежать ошибки SyntaxError

  File "d:\Dev\Code\ITClub\Python\PY02\les03\hw.py", line 25
    print(x/)
            ^
SyntaxError: invalid syntax

Пример:

a = 1
b = 2
c = 3

print("Исходные числа:", a, b, c)  # кавычки в начале и конце строки
                                   # аргументы через запятую

average = (a + b + c) / 3

print("Среднее значение", average)  # Круглые скобки для вызова функции print

Результат функции и сохранение в переменной

Используем присваивание для сохранения результата выполнения функции в переменной

Пример:

result = max(1, 2, 3)  # Результат функции max сохраняется в переменной result
