Повтор материала

Прежде чем двигаться дальше, давай убедимся, что ты понимаешь основные термины. Если что-то непонятно, не стесняйся задавать вопросы!

  • Литеральные значения — это конкретные значения, которые ты используешь в коде, например, число 5 или строка "Привет".
  • Операторы — это символы, которые выполняют операции над значениями, например, + (сложение) или == (проверка равенства).
  • Переменные — это "контейнеры" для хранения данных. Например, x = 10 создаёт переменную x и присваивает ей значение 10.
  • Функции — это блоки кода, у которого есть имя, и который выполняют определённую задачу. Например, print("Привет") выводит текст на экран.
  • Выражения — это комбинации значений, переменных и операторов, которые вычисляются в одно значение. Например, x + 5 — это выражение.
  • Ключевые слова — это специальные слова в Python, которые имеют определённое значение и не могут быть использованы как имена переменных. Например, if, else, for.
  • Управляющие конструкции — это конструкции, которые управляют выполнением программы, например, if, for, while.
  • Комментарии — это текст в коде, который игнорируется Python. Они начинаются с символа # и используются для объяснения кода.

Значения функций

Функции могут возвращать значения. Например, функции round, len или input. Это значение можно присвоить переменной:

# Присвоение результата функции переменной
number = round(3.14159, 2)  # number будет равно 3.14
print(number)

length = len("Привет!")     # length будет равно 7
print(length)

user_input = input("Введите что-нибудь: ")  # Сохраняем введённое значение
print("Вы ввели:", user_input)

Возвращённое значение можно также использовать сразу, без сохранения в переменной:

print(round(3.14159, 2))  # Выведет 3.14
print(len("Привет!"))     # Выведет 7
print(input("Введите что-нибудь: "))  # Выведет введённое пользователем значение

Упражнение: вывод строк и чисел

Использование print с разным количеством аргументов

Функция print может выводить на экран несколько значений одновременно. Например:

print("Привет", "мир", "!")  # Выведет: Привет мир !

Конкатенация строк

Конкатенация — это сложение строк. Например:

name = "Аня"
greeting = "Привет, " + name + "!"
print(greeting)  # Выведет: Привет, Аня!

Комбинирование строк и чисел при помощи str

Чтобы вывести число вместе с текстом, нужно преобразовать число в строку с помощью функции str:

age = 13
print("Мне " + str(age) + " лет.")  # Выведет: Мне 13 лет.

F-строки

F-строки — это удобный способ вставлять значения переменных в строку. Например:

name = "Аня"
age = 13
print(f"Меня зовут {name} и мне {age} лет.")  # Выведет: Меня зовут Аня и мне 13 лет.

Переносы строки и тройные кавычки

Если нужно вывести текст с переносами строк, можно использовать тройные кавычки:

text = """Привет!
Это многострочный текст.
Ура!"""
print(text)

Управляющие конструкции

Оператор if

Оператор if позволяет выполнять код только при выполнении определённого условия. Например:

age = 13
if age >= 12:
    print("Ты можешь смотреть фильмы с рейтингом PG-13!")

Если условие не выполняется, можно использовать оператор else, чтобы выполнить альтернативный блок кода:

age = 10
if age >= 12:
    print("Ты можешь смотреть фильмы с рейтингом PG-13!")
else:
    print("Прости, ты ещё слишком мал для фильмов с рейтингом PG-13.")

Также можно добавить дополнительные условия с помощью elif:

age = 18
if age < 12:
    print("Ты можешь смотреть только фильмы с рейтингом G или PG.")
elif age < 18:
    print("Ты можешь смотреть фильмы с рейтингом PG-13!")
else:
    print("Ты можешь смотреть фильмы с любым рейтингом!")

В итоге конструкции if, elif и else позволяют создавать сложные ветвления логики в зависимости от разных условий.

Диаграмма для этого примера:

Операторы сравнения

Операторы сравнения используются для сравнения значений:

  • == — равно
  • != — не равно
  • > — больше
  • < — меньше
  • >= — больше или равно
  • <= — меньше или равно

Пример:

x = 10
y = 20
if x < y:
    print("x меньше y")
