Что мы узнали

Код Python состоит из комбинации основных строительных блоков, включающих:

  • литеральные значения
  • операторы
  • переменные
  • функции
  • ключевые слова

Вместе эти элементы формируют выражения и инструкции (команды), которые составляют основу любой программы на Python.

Объяснение:

  1. Литеральные значения: Это фиксированные данные или константы в коде, такие как числа (42, 3.14), строки ("Привет, мир!") или логические значения (True, False).

    • Пример: 
      10  # Литеральное значение, представляющее число
"Hello, Python!"  # Литеральное значение, представляющее строку

  2. Операторы: Символы или ключевые слова, которые выполняют операции над значениями или переменными, например, арифметические (+, -), сравнения (==, >) или логические (and, or) операторы.

    • Пример: 
      5 + 3  # Складывает два числа
x > 10  # Сравнивает переменную с числом

  3. Переменные: Служат для хранения данных и имеют имя, чтобы их можно было повторно использовать или изменять позже.

    • Пример: 
      age = 25  # Переменная с именем 'age', содержащая значение 25
          # Оператор = присваивает значение переменной

  4. Функции: Повторно используемые блоки кода, которые выполняют определенные задачи. Например, print() предназначена для вывода текста на экран. Вы можете создавать новые функции.

    • Пример: 
      def greet():
    print("Привет!")
greet()  # Вызывает функцию и выводит "Привет!"

  5. Ключевые слова: Зарезервированные слова в Python, имеющие особое значение, такие как if, else, while, for, def. Их нельзя использовать как имена переменных.

    • Пример: 
      if age > 18:
    print("Вы совершеннолетний!")

Почему это важно:

Python объединяет эти элементы для создания выражений (которые вычисляют значения) и инструкций или команд (которые выполняют действия). Понимание того, как эти части работают вместе, является ключом к эффективному написанию кода на Python.

Выражения (Expressions)

Выражения - это комбинации значений, переменных и операторов, которые Python посчитать и вернуть результат.

  1. Простые арифметические (математические) выражения:

    5 + 3  # Складывает 5 и 3, возвращает результат 8

  2. Использование переменных:

    x = 10
y = 20
x + y  # возвращает 30

  3. Вызов функций в выражениях:

    len("Python")  # возвращает 6 (длина строки)

  4. Булевые (Boolean) выражения:

    10 > 5  # возвращает True

Команды (Statements)

Команды - это инструкции или действия, которые выполняет Python, например присваивание переменной значения, вызов функции для вывода на экран или выполнение цикла.

  1. Команда присваивания:

    x = 42  # присваивает значение 42 переменной x

  2. Вызов функции print для вывода на экран:

    print("Hello, Python!")  # выводит "Hello, Python!" на консоль

  3. Условный оператор:

    if x > 10:
    print("x больше 10")  # выполняется если условие истинно (True)

  4. Команда создания (определения) функции:

    def greet():
    print("Hello!")
greet()  # вызвает функцию, которая печатает "Hello!"

  5. Команда цикла (повторов):

    for i in range(3):
    print(i)  # печатает 0, 1, 2

Разница между выражениями и командами

  • Выражения: их можно вычислить и они возвращают результат.

    • Пример:

      5 * 3  # результат выражения: 15

      Выражения могут использоваться внутри команд.

  • Команда: выполняет действие, и не всегда возвращает результат.

    • Example: 
      print(5 * 3)  # команда выводит результат выражения (15)

Выражения нужны чтобы посчитать значение, а команды определяют действия, которые Python должен выполнить. Нужно понимать и то и другое, чтобы разбираться в коде Python и писать без ошибок!

Типы данных в Python

1. Каждое значение и переменная имеют тип

В Python каждое значение (например, числа, строки или объекты) имеет определённый тип, а переменная наследует тип значения, которое она хранит. Тип определяет, как ведёт себя значение и какие операции можно с ним выполнять.

  • Пример: 
    x = 42          # x имеет тип int
y = 3.14        # y имеет тип float
name = "Alice"  # name имеет тип str
print(type(x))  # Вывод: <class 'int'>
print(type(name))  # Вывод: <class 'str'>

2. Операторы зависят от типа

Операторы Python (такие как +, -, *, /) работают по-разному в зависимости от типа значений, к которым они применяются. Некоторые операторы допустимы только для определённых типов, а другие могут вести себя по-разному в зависимости от типа операндов.

  • Пример: Арифметические операторы: 
    a = 10
b = 3.5
c = "Hello"

# Арифметические операции
print(a + b)  # Вывод: 13.5 (int + float = float)
print(a * 2)  # Вывод: 20 (int * int = int)

# Конкатенация строк
print(c + " World")  # Вывод: "Hello World"

3. Ошибки при несоответствии типа и оператора

Если вы попытаетесь применить оператор к значению неподходящего типа, Python вызовет ошибку TypeError. Это гарантирует, что код не выполняет некорректные или неопределённые операции.

  • Пример недопустимых операций:

    a = 10
b = "Python"

# Ошибка: Сложение числа и строки
print(a + b)  
# Вывод: TypeError: unsupported operand type(s) for +: 'int' and 'str'

  • Пример исправления ошибок с помощью преобразования типов:

    a = 10
b = "20"

# Преобразование строки в int для выполнения арифметики
print(a + int(b))  # Вывод: 30

# Преобразование числа в строку для конкатенации
print(str(a) + b)  # Вывод: "1020"

Подводим итог

  1. Каждое значение и переменная в Python имеют тип, который можно проверить с помощью функции type().
  2. Операторы, такие как +, -, *, и /, работают по-разному в зависимости от типа операндов. Например:
    • + складывает числа, но конкатенирует строки.
    • * умножает числа, но повторяет списки или строки.
  3. Если оператор применяется к несовместимым типам, Python вызывает ошибку TypeError.
  4. Чтобы избежать ошибок, используйте преобразование типов (например, int(), str(), float()) для приведения значений к совместимым типам.

Эти принципы подчёркивают, как Python динамически управляет типами и обеспечивает осмысленность операций.

Что делать, если при запуске кода возникает ошибка?

Ошибки при выполнении кода — это нормальная часть процесса программирования. Вместо того чтобы бояться ошибок, важно научиться их анализировать и исправлять. Вот пошаговое руководство:

1. Прочитайте текст ошибки

Python предоставляет подробное описание ошибки, называемое "трейсбеком" (traceback). Он включает:

  • Тип ошибки (например, SyntaxError, TypeError, NameError).

  • Сообщение об ошибке, поясняющее, что пошло не так.

  • Номер строки, где произошла ошибка.

  • Пример ошибки:

    print("Hello
    • Traceback (отслеживание вызовов): 
      File "example.py", line 1
    print("Hello
               ^
SyntaxError: EOL while scanning string literal
    • Сообщение указывает, что строка закончилась внезапно (EOL), что говорит о незакрытых кавычках.

2. Понять тип ошибки

Часто встречающиеся ошибки:

  • SyntaxError: Ошибка синтаксиса. Например, пропущенные скобки или двоеточие.

    • Пример: 
      if x > 10
    print(x)
      • Исправление: 
        if x > 10:
    print(x)

  • NameError: Использование неопределённой переменной.

    • Пример: 
      print(x)
      • Исправление: Убедитесь, что переменная определена до использования. 
        x = 10
print(x)

  • TypeError: Оператор или функция применяются к несоответствующему типу данных.

    • Пример: 
      print("Age: " + 25)
      • Исправление: 
        print("Age: " + str(25))

  • IndexError: Попытка обратиться к несуществующему элементу в списке.

    • Пример: 
      items = [1, 2, 3]
print(items[5])
      • Исправление: 
        print(items[2])  # Последний элемент

3. Проверьте строку кода

  • Посмотрите на строку, упомянутую в ошибке.
  • Проверьте соседние строки: проблема может быть вызвана кодом выше или ниже.

4. Используйте отладку

  • Вывод значений: Добавьте временные команды print() для проверки значений переменных.

    • Пример: 
      x = 10
y = 0
print("Before division")  # Для отладки
print(x / y)
      • Исправление: 
        if y != 0:
    print(x / y)

  • Интерактивная оболочка (REPL): Запустите проблемный код в консоли Python, чтобы проверить его поэтапно.

5. Ищите решение

Если ошибка остаётся непонятной:

  • Гуглите текст ошибки. Например: “TypeError unsupported operand types for +”.
  • Обращайтесь к документации Python.
  • Используйте форумы, такие как Stack Overflow.

6. Исправляйте и тестируйте

  • После внесения исправлений тестируйте код снова.
  • Вносите изменения по одному и проверяйте их работу, чтобы изолировать проблему.

7. Профилактика ошибок

  • Следите за синтаксисом: Используйте IDE, такие как VS Code или PyCharm, которые подсвечивают ошибки.
  • Разбивайте код на части: Пишите и тестируйте небольшие блоки кода.
  • Документируйте код: Комментарии помогут понять логику позже.
    • Пример: 
      # Проверяем деление на 0
if y != 0:
    print(x / y)

Ошибки — это не проблема, а возможность учиться и совершенствоваться!

Дополнительная информация

Вот список операторов в Python, сгруппированных по категориям:

1. Арифметические операторы

Используются для математических операций.

Оператор Описание Пример Результат
+ Сложение 5 + 3 8
- Вычитание 5 - 3 2
* Умножение 5 * 3 15
/ Деление 5 / 3 1.666...
// Целочисленное деление 5 // 3 1
% Остаток от деления 5 % 3 2
** Возведение в степень 2 ** 3 8

2. Операторы сравнения (реляционные)

Используются для сравнения значений, возвращают True или False.

Оператор Описание Пример Результат
== Равно 5 == 3 False
!= Не равно 5 != 3 True
> Больше 5 > 3 True
< Меньше 5 < 3 False
>= Больше или равно 5 >= 3 True
<= Меньше или равно 5 <= 3 False

3. Логические операторы

Используются для объединения условных выражений.

Оператор Описание Пример Результат
and Логическое И True and False False
or Логическое ИЛИ True or False True
not Логическое НЕ not True False

4. Операторы присваивания

Используются для присваивания значений переменным.

Оператор Описание Пример Эквивалентно
= Присваивание x = 5 x = 5
+= Сложение с присваиванием x += 3 x = x + 3
-= Вычитание с присваиванием x -= 3 x = x - 3
*= Умножение с присваиванием x *= 3 x = x * 3
/= Деление с присваиванием x /= 3 x = x / 3
//= Целочисленное деление с присваиванием x //= 3 x = x // 3
%= Остаток от деления с присваиванием x %= 3 x = x % 3
**= Возведение в степень с присваиванием x **= 3 x = x ** 3

5. Побитовые операторы

Работают на уровне битов.

Оператор Описание Пример Результат
& И 5 & 3 1
` ` ИЛИ `5
^ Исключающее ИЛИ 5 ^ 3 6
~ НЕ ~5 -6
<< Сдвиг влево 5 << 1 10
>> Сдвиг вправо 5 >> 1 2

6. Операторы вхождения

Проверяют, содержится ли значение в коллекции.

Оператор Описание Пример Результат
in Содержится в коллекции 'a' in 'apple' True
not in Не содержится 'x' not in 'apple' True

7. Операторы идентичности

Проверяют, являются ли два объекта одним и тем же.

Оператор Описание Пример Результат
is Тот же объект x is y True/False
is not Не тот же объект x is not y True/False

8. Специальные операторы

  • Тернарный условный оператор: 
    result = "Да" if x > 0 else "Нет"
