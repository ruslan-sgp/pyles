Технологии front end и back end

Front end — это часть приложения, которую видит пользователь. В веб-приложении это всё, что происходит в браузере: внешний вид сайта, кнопки, формы, анимации. Основные технологии для front end — это HTML (структура страницы), CSS (стили и оформление) и JavaScript (интерактивность).

Back end — это "внутренняя" часть приложения, которая работает на сервере. Она отвечает за обработку данных, хранение информации в базах данных и выполнение сложных вычислений. Back end может быть написан на разных языках, включая Python, Java, PHP и другие.

Пример:
Когда вы заходите на сайт, front end — это то, что вы видите: кнопки, текст, картинки. А back end — это то, что происходит "за кулисами": например, когда вы вводите логин и пароль, back end проверяет их правильность в базе данных.

Front End также называют клиентскими приложениями, Back End называют серверными приложениями.

Популярные языки программирования

Языки программирования — это инструменты, которые помогают создавать программы.

Вот несколько популярных языков:

  1. Python — простой и мощный язык общего назначения, который используется для создания сайтов, анализа данных, игр и многого другого.
  2. JavaScript — язык для создания интерактивных веб-страниц. Он работает в браузере и делает сайты "живыми". Простой и наглядный в изучении.
  3. Java — язык для создания серверных и клиентских приложений на разных ОС (операционных системах). Minecraft создан на Java.
  4. C++ — язык для создания высокопроизводительных программ, таких как игры или операционные системы.
  5. C# - язык для создания клиентских и серверных приложений для платформы Windows и других ОС; используется для разработки игр в Unity.

Отчет GitHub Octoverse показывает, какие языки программирования самые популярные среди разработчиков.

После изучения этого курса ты сможешь понимать эти языки. Это поможет тебе создать свои сайты, игры или приложения! Ты сможешь разрабатывать что-то уникальное, как делают профессиональные программисты.

Создание и запуск программы Python

А. В среде VS Code

  • Создайте папку для ваших проектов
    • Совет: Используйте английские буквы в названиях папок и файлов. Избегайте пробелов и тире, чтобы избежать ошибок.
    • Папку можно создать при помощи Windows Explorer или при помощи панели Explorer в VS Code
  • Откройте эту папку (не отдельный файл) при помощи VS Code
  • Создайте новый файл с расширением .py (например, hello.py)
  • Добавьте код, например:
print("Привет, мир!")
  • Сохраните файл (Ctrl + S).
  • Запустите программу, нажав на кнопку "Run" (иконка с треугольником в правом верхнем углу) или используя сочетание клавиш Ctrl + F5.
    • Данная функция доступна после установки расширения Python для VS Code
  • Результат появится в окне терминала.

Б. В интерпретаторе командной строки

Интерпретатор Python позволяет выполнять код прямо в командной строке. Вот как это сделать:

  1. Откройте командную строку (Terminal на Mac/Linux или Command Prompt на Windows).
  2. Введите команду py чтобы запустить интерпретатор (в более старых версиях - команда python или python3).
  3. Введите код Python, например: 
    print("Привет из интерпретатора REPL")
  4. Нажмите Enter чтобы выполнить команду
  5. Результат появится ниже:

Интерпретатор также иногда называют REPL, что значит Read-Eval-Print Loop. Это означает, что вы можете вводить команды, выполнять их, а затем видеть результат прямо в командной строке.

Функция print

Функция print — это одна из самых простых и полезных функций в Python. Она выводит текст или данные на экран.

Примечание: В Python называния функций и переменных "чувствительны к регистру букв". Это значит, что названия и ключевые слова нужно писать так как указано - не менять большие буквы на маленькие или наоборот. Например, функция print пишется с маленькой буквы.

Пример:

print("Привет, мир!")

Этот код выведет на экран текст "Привет, мир!".

Аргументы функции — это данные, которые мы передаем в функцию. В примере выше аргумент — это строка "Привет, мир!".

Термины и понятия

  • Веб-приложения - приложения которые используют Интернет для работы
  • Front end — видимая часть приложения, с которой взаимодействует пользователь.
  • Back end — "внутренняя" часть приложения, которая обрабатывает данные.
  • Запрос (request) — обращение клиента (например, браузера) к серверу для получения данных.

Технологии front end:

  • HTML — язык для создания структуры веб-страниц.
  • CSS — язык для оформления и стилизации веб-страниц.
  • JavaScript — язык для добавления интерактивности на веб-страницы.

Технологии back end:

  • Сервис (service) — программа или часть программы, которая выполняет определенные задачи.
  • База данных (database) — хранилище информации, где данные организованы и легко доступны.
  • API — интерфейс для взаимодействия между программами. Например, API позволяет сайту получать данные от сервера.

Термины языков программирования:

  • Функция — часто используемый блок кода, который выполняет определенную задачу. Например, print — это функция, которая выводит данные на экран.
  • Аргументы — данные, которые передаются в функцию для выполнения задачи.

Домашнее задание

Напишите программу, которая выполняет математические операции над числами и выводит результат.

Пример:

x = 25
y = 8

# Сложение
print("Сумма:", x + y)

# Вычитание
print("Разность:", x - y)

# Умножение
print("Произведение:", x * y)

# Деление
print("Частное:", x / y)

Задание:

  1. Добавьте операции возведения в степень (**) и нахождения остатка от деления (%).
  2. Попробуйте использовать другие числа.
  3. Выведите результаты всех операций в одной программе.
