Пузырьковая сортировка - Bubble Sort

Этот код реализует алгоритм сортировки пузырьком, который является простым методом сортировки. Он многократно проходит по списку, сравнивает соседние элементы и меняет их местами, если они расположены в неправильном порядке. Процесс повторяется до тех пор, пока список не будет отсортирован.

Разбор кода

#include <iostream>
  • Библиотека iostream
  • Объяснение: Эта строка включает стандартную библиотеку ввода-вывода в C++. Она позволяет программе использовать cout для вывода и cin для ввода.
using namespace std;
  • Пространство имен std
  • Объяснение: Эта строка позволяет программе использовать все сущности в пространстве имен std без необходимости добавлять префикс std::. Например, можно использовать cout вместо std::cout.
  • Иллюстрация:
void bubbleSort(int arr[], int n) {
  • Функция bubbleSort
  • arr[] - входной массив
  • n - размер массива
  • Объяснение: Эта строка определяет функцию с именем bubbleSort, которая принимает массив целых чисел (arr) и его размер (n) в качестве параметров. Функция будет сортировать массив в порядке возрастания.
    for (int i = 0; i < n-1; i++) {
  • Внешний цикл (i = 0 до n-2)
  • Объяснение: Этот внешний цикл выполняется от 0 до n-2. Он представляет собой каждый проход по массиву. Каждый проход гарантирует, что наибольший неотсортированный элемент перемещается на свое правильное место.
        for (int j = 0; j < n-i-1; j++) {
  • Внутренний цикл (j = 0 до n-i-2)
  • Объяснение: Этот внутренний цикл выполняется от 0 до n-i-2. Он сравнивает соседние элементы в массиве. По мере увеличения i количество сравнений уменьшается, потому что последние i элементов уже отсортированы.
            if (arr[j] > arr[j+1]) {
  • Сравнить arr[j] и arr[j+1]
  • Объяснение: Это условное выражение проверяет, больше ли текущий элемент (arr[j]) следующего элемента (arr[j+1]). Если это так, это указывает на то, что они расположены в неправильном порядке и должны быть обменяны местами.
                int temp = arr[j];
                arr[j] = arr[j+1];
                arr[j+1] = temp;
  • Перестановка элементов, если они не расположены по порядку
  • Объяснение: Эти три строки выполняют операцию обмена. Временная переменная (temp) используется для хранения значения arr[j], что позволяет нам перезаписать его значением arr[j+1], а затем присвоить temp значению arr[j+1].

После завершения всех проходов массив будет отсортирован.

Главная функция

int main() {
  • Объяснение: Эта строка определяет главную функцию, с которой начинается выполнение программы. 
    int arr[] = {95, 10, 75, 12, 64, 34, 25, 12, 22, 11, 90};
  • Объяснение: Инициализируется массив целых чисел с именем arr, содержащий набор неотсортированных целых чисел.
    int n = sizeof(arr)/sizeof(arr[0]);
  • Объяснение: Эта строка вычисляет количество элементов в массиве, деля общий размер массива на размер одного элемента. Это значение сохраняется в переменной n.
    bubbleSort(arr, n);
  • Объяснение: Вызывается ранее определенная функция bubbleSort с параметрами arr и его размером n, сортируя массив на месте. 
    cout << "Отсортированный массив: \n";
  • Объяснение: Эта строка выводит сообщение о том, что далее будет отсортированный массив.
    for (int i = 0; i < n; i++) {
        cout << arr[i] << " ";
    }
  • Объяснение: Этот цикл проходит по каждому элементу отсортированного массива и выводит его с пробелом.
    return 0;
}
  • Объяснение: Эта строка указывает на то, что программа завершена успешно и возвращает управление операционной системе.

Отсортированный массив:
