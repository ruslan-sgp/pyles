Компьютер - более навороченный калькулятор!

Компьютер — это, по сути, продвинутый калькулятор, который умеет не только считать, но и управлять информацией, графикой и процессами. (на самом деле, английские слова compute и calculate - синонимы)

В чём они отличаются?

На калькуляторе:

  • вы вводите числа
  • вводите операторы (или функции, такие как log или sin)
  • нажимаете =
  • он считает и показывает результат
  • можно запомнить результаты (кнопка M, англ. Memory) и использовать их в дальнейших расчетах

На компьютере:

  • вы пишете код на языке программирования и сохраняете его в файле
  • компьютер понимает не только числа, но и текст, и логические значения, и другое
  • больше разных операторов
  • намного больше функций, и можно добавлять свои
  • есть управляющие команды: например проверки условия - if, циклов - for, и другие
  • для запоминания результатов используются переменные
  • есть и другие конструкции, например комментарии

Рассмотрим подробнее как устроены программы на Python.

Как устроен код Python

Значения (данные)

Помимо чисел, Python поддерживает и другие типы данных:

  • текст (строки). Заключаются в кавычки.
  • логические значения: True и False
  • тип None (пустое значение)
  • коллекции - списки, кортежи и словари
  • объекты

В конце этого конспекта типы описаны более подробно.

Операторы

Операторы - это действия, которые можно выполнить над значениями. Например, сложение, вычитание, умножение, деление, и другие. В Python есть много разных операторов.

В зависимости от типа данных, можно применять различные операторы, и они могут работать по-разному.

Например, сравни чему равны следующие выражения:

# сложение чисел
1 + 2

# сложение строк
"1" + "2"

Выражения

Выражения используются чтобы что-то посчитать. Примеры выражений:

(1 + 2) ** 3          # равно 27
123 * 456 > 55_000    # равно True

Вопрос: а как написать следующие выражения на Python?

Переменные

Переменные используются чтобы сохранить значения и использовать их позже. Например:

# присваиваем значение 5 переменной x
x = 5

# присваиваем значение выражения переменной y
y = 123 * 45

# используем значение x
print(x)      # выводит 5

# используем значение y
print(y)      # выводит 5535

В отличие от строк, имена переменных записываются без кавычек.

Управляющие конструкции

Позволяют изменить ход выполнения программы. Например, if выполняет действие в зависимости от условия, а for позволяет повторить действие несколько раз.

# если x больше 100, то выводим сообщение
if x > 100:
  print("x больше 100")

# повторяем действие 5 раз
for i in range(5):
  print(i)

Для записи управляющих конструкций используются ключевые слова. Они записываются без кавычек, и их нельзя использовать в качестве имён переменных.

Функции

Функции - это блок кода Python (то есть набор команд), у которого есть имя. Функции можно вызывать, чтобы этот код выполнился. Для этого нужно добавить скобки к имени функции. У функций могут быть аргументы. Например:

# функция print
# вызвать без аргументов
print()          # выводит пустую строку

# вызвать с аргументом
print("hello")

# вызвать с двумя аргументами
print("Два плюс два равно", 4)

Можно создавать свои функции:

# функция, которая выводит приветствие
def say_hello():
  print("Привет!")

# вызываем функцию
say_hello()  # выводит "Привет!"

Имена функций записываются без кавычек.

Дополнительный материал

Примеры разных типов данных

1. Текст (строки)

Строки — это текст. Это может быть имя, сообщение или что-то ещё. Строки заключаются в кавычки: ', ", или """ для длинного текста.
Пример:

# Примеры строк
name = "Alex"
greeting = 'Hello, world!'
long_text = """This is
a really long
piece of text."""

print(name)  # Вывод: Alex
print(greeting)  # Вывод: Hello, world!

2. Логические значения (True и False)

True означает "Да", а False — "Нет". Эти значения полезны для проверки условий.
Пример:

# True и False
is_sunny = True
is_raining = False

if is_sunny:
    print("На улице солнечно! Идём гулять!")
else:
    print("Лучше остаться дома.")

3. None (пустое значение)

None — это как пустая коробка. Оно означает "ничего".
Пример:

# Пример None
gift = None

if gift is None:
    print("Подарка пока нет!")
else:
    print("Смотри, что я получил:", gift)

4. Коллекции: списки, кортежи и словари

Эти структуры данных помогают хранить группы элементов.

Списки

Списки — это как рюкзак: туда можно добавлять, убирать или менять вещи.
Пример:

# Пример списка
backpack = ["notebook", "pen", "sandwich"]
backpack.append("water bottle")  # Добавляем элемент
print(backpack)  # ['notebook', 'pen', 'sandwich', 'water bottle']

Кортежи

Кортежи — это как запечатанный сундук: можно посмотреть, что внутри, но менять нельзя.
Пример:

# Пример кортежа
treasures = ("gold", "jewels", "map")
print(treasures[0])  # Вывод: gold

Словари

Словари — это как телефонная книга: у каждой записи есть имя (ключ) и значение.
Пример:

# Пример словаря
phone_book = {
    "Alice": "123-4567",
    "Bob": "987-6543",
}

print(phone_book["Alice"])  # Вывод: 123-4567
phone_book["Charlie"] = "555-6789"  # Добавляем новую запись
print(phone_book)  # {'Alice': '123-4567', 'Bob': '987-6543', 'Charlie': '555-6789'}

5. Объекты (составные или сложные типы)

Объекты - это модели существующих в жизни вещей. Другие типы данных (строки, числа, и пр.) тоже можно рассматривать как объекты. У объектов есть свойства (данные) и методы (то, что они умеют делать).
Пример:

# Объект "Книга"
book.title = "Гарри Поттер и философский камень"
book.author = "Дж. К. Роулинг"
book.year = 1997
